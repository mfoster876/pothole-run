# Pothole Run

A pixel-art endless dodge-runner with a Jamaican soul: roll a downtown-Kingston
makeshift **handcart** — sound system blaring on the back — down Jamaica's
notoriously potholed roads, sliding between lanes to dodge craters, traffic and
goats for as long as your rotten cart holds together.

- **Play:** https://mfoster876.github.io/pothole-run/
- **Controls:** hold left / right (touch) or arrow keys / A,D. `M` mutes.
- **Built with:** vanilla HTML5 Canvas + ES modules, no build step, no dependencies.
- **Run tests:** `node --test`

Phase 1: 3 drivers (School Yute, Rasta Musician, reckless Bleachaz Conductor),
3 stages (Fern Gully, Holland Bamboo, Negril 7-Mile), cart-wreck survival, coins,
on-device save, installable PWA. See `docs/superpowers/` for the full design.
